## EMIASQueueBot (Python-версия)

Этот бот управляет очередью разработчиков в Telegram‑канале с топиками (iOS / Android).  
Очередь **отдельная для каждого топика**. Данные хранятся в файле `queues.json` рядом с ботом.

### Требования

- Установленный **Python 3.9+** (желательно 3.11+)
- Доступ в интернет с машины, где будет крутиться бот
- Токен Telegram‑бота от BotFather

### Подготовка

1. Склонировать репозиторий (или скопировать папку проекта) на машину.
2. Войти в директорию проекта:

```bash
cd /путь/до/EMIASQueueBot
```

3. Создать и активировать виртуальное окружение Python:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

4. Установить зависимости:

```bash
pip install -r requirements.txt
```

5. Задать переменную окружения с токеном бота:

```bash
export EMIAS_QUEUE_BOT_TOKEN=ТОКЕН_ОТ_BOTFATHER
```

> Токен вида `123456789:ABCDEF...`. Не коммитить его в git.

### Запуск бота

Находясь в каталоге проекта и с активированным виртуальным окружением:

```bash
source .venv/bin/activate      # если ещё не активировано
export EMIAS_QUEUE_BOT_TOKEN=ТОКЕН_ОТ_BOTFATHER
python3 bot.py
```

В терминале должно появиться сообщение:

```text
Python EMIASQueueBot started
```

Бот будет работать, пока запущен этот процесс (можно остановить `Ctrl + C`).

### Настройка в Telegram

1. Добавить бота в нужный канал/группу, где включены **топики (форум)**.
2. Создать/использовать два топика, например:
   - `iOS`
   - `Android`
3. В каждом топике можно использовать одни и те же команды — бот различает их по `message_thread_id`, поэтому очереди **раздельные**.

### Команды бота

Все команды выполняются **внутри нужного топика** (iOS или Android):

- **`/add`**  
  Добавить себя в очередь этого топика.  
  Если пользователь уже в какой‑либо очереди (любого топика), бот ответит «Вы уже в очереди».

- **`/list`**  
  Показать текущую очередь в этом топике.

- **`/remove`**  
  Удалить себя из очереди этого топика.

- **`/clear`** *(только админ чата)*  
  Очистить очередь в этом топике.

- **`/remove_user @username`** *(только админ чата)*  
  Удалить указанного пользователя из очереди этого топика по нику.  
  Удобнее всего вводить так:

  ```text
  /remove_user @
  ```

  затем выбрать человека из списка упоминаний и отправить.

### Хранение очереди и поведение при перезапуске

- Все очереди хранятся в файле `queues.json` в корне проекта.
- При каждом изменении (`/add`, `/remove`, `/clear`, `/remove_user`) файл перезаписывается.
- Если машина выключится или процесс бота упадёт:
  - при следующем запуске `python3 bot.py` бот загрузит очереди из `queues.json`;
  - команды, отправленные **во время простоя**, не обрабатываются задним числом.

### Фоновый запуск (опционально)

На сервере/VPS бота можно запускать не в интерактивном терминале, а через systemd, tmux, screen или другой менеджер процессов. Основная команда остаётся той же:

```bash
cd /путь/до/EMIASQueueBot
source .venv/bin/activate
export EMIAS_QUEUE_BOT_TOKEN=ТОКЕН_ОТ_BOTFATHER
python3 bot.py
```
